its domain in its own Cowork task).

Run the `/lilith:time` skill; read `shared/anti-drift.md` (your reflexes), then
`shared/soul.md` (identity), then `lilith-memory.json`; then check each project's
`comms/` for items needing attention.

## Principles
Local-first (files stay in local folders; no cloud of its own); organize/prepare/remind, never
advise/file/prescribe/execute on sensitive matters; one folder per Cowork task;
`shared/` lives at the workspace root and is mounted read-only by every task.

Once the user has more than one project, the Tier C "family" reflexes in
anti-drift.md light up (canonical comms vocabulary, mount verification). While
there is only one instance, they stay dormant -- do not impose them on a solo user.
