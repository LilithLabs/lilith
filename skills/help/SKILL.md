# Help

Briefly explain Lilith (a staff, not a chatbot; local-first) and list the
available skills with one line each and a trigger example:

- `genesis` -- first-time setup. "Set me up."
- `tutorial` -- guided walk-through, including opening project tasks. "Walk me through it."
- `time` -- reliable date/time.
- `soul` -- author your Lilith's identity. "Let's set your soul."
- `rebirth` -- continuity letters + birth new projects. "Birth a project."
- `brief` -- a status briefing across your day/projects.
- `comms-check` -- scan for items between your daemons.
- `roster` -- list and manage your projects.
- `consolidate-memory` -- tidy your memory files.
- `schedule` -- set up recurring briefs/checks.

If the user names a goal, point them at the single best skill rather than
listing everything.
